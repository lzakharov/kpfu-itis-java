Игроки могут перемещаться вправо/влево/вверх/вниз и стрелять. Попадание в другого игрока ведет к его смерти.

Обмен информацией по сети с помощью UDP протокола.
Информация делиться на 4 вида передаваемых пакетов:
	LoginPacket - содержит в себе username отправителя и отправляется при подключении к серверу
	DisconnectPacket - содержит в себе username отправителя и отправляется при отключении от сервера(выхода из игры)
	MovePacket - содердит в себе username отправителя, координаты нового положения и направление движения
	ShootPacket - содержит в себе username, координаты отправителя и направление выстрела

Пакеты отличают друг от друга по префиксу - первому байту пакета. 
Формат пакетов:
	LoginPacket: 0 + username
	DisconnectPacket: 1 + username
	MovePacket: 2 + username + "," + x + "," + y + "," + dir
	ShootPacket: 3 + username + "," + x + "," + y + "," + dir
